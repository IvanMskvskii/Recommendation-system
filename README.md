# Recommendation_system

Проект посвящён задаче построения рекомендательной системы на основе явной обратной связи - рейтингов объектов (explicit feedback).
Целью проекта является обучение модели, способной рекомендовать пользователям наиболее релевантные фильмы на основе их предыдущих взаимодействий.

Рекомендательные системы были реализованы на основе двух библиотек: Cornac и LightFM. Cornac позволила провести эксперименты с разными моделями и метриками, в то время как LightFM реализует один из лучших алгоритмов матричной факторизации. 

В качестве основной модели используется библиотека LightFM, позволяющая эффективно работать с разреженными данными. В ходе работы были подготовлены и вузализированы данные, обработаны контентные признаки, исключены объекты с малым количеством взаимодействий. В результате была сформирована матрица взаимодействий пользователей и объектов, а также проведено разделение данных на тренировочную и тестовую выборки.

В рамках проекта были обучены модели с различными гиперпараметрами, проведена оценка качества рекомендаций и проанализировано влияние параметров модели на результаты.

Основная часть проекта выполняется в локальном окружении.
Для обеспечения воспроизводимости и корректной установки зависимостей, необходимых для обучения рекомендательной модели на базе LightFM, соответствующая часть проекта была вынесена в Docker-контейнер.

Некоторые элементы ноутбука используют ipywidgets и могут некорректно отображаться в веб-интерфейсе GitHub. Для корректного просмотра запускайте ноутбук локально.

В заключении проекта рассмотрены возможные направления для дальнейшего улучшения рекомендательной системы, включая добавление признаков пользователей и объектов, увеличение количества данных, а также эксперименты с альтернативными моделями.
